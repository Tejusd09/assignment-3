This repository contains solutions for two Python programming tasks as part of Module 4: Functions & Modules in Python.

Task 1: Factorial Calculator (task1_factorial.py)

Description
A Python program that calculates the factorial of a number using a custom function with a loop.

Features:
- Defines a factorial() function that takes a number as argument
- Uses iterative approach (for loop) to calculate factorial
- Handles edge cases: negative numbers and zero
- Shows calculation process for numbers up to 10
- Includes comprehensive error handling

How to Run:
python task1_factorial.py

Sample Output:
Factorial Calculator
-------------------
Enter a non-negative integer: 5

Factorial of 5 is: 120
Calculation: 1 × 2 × 3 × 4 × 5 = 120

Task 2: Math Module Calculations (task2_math_module.py)

Description
A Python program that uses Python's built-in math module to perform various mathematical calculations.

Features:
- Uses math.sqrt() for square root calculation
- Uses math.log() for natural logarithm
- Uses math.sin() for sine calculation in radians
- Handles edge cases for invalid operations
- Includes additional trigonometric calculations
- Displays results formatted to 4 decimal places

How to Run:
python task2_math_module.py

Sample Output:
Math Module Calculations
-----------------------
Enter a number: 25

Calculations for number: 25.0
----------------------------------------
Square root: √25.0 = 5.0000
Natural logarithm: ln(25.0) = 3.2189
Sine (in radians): sin(25.0) = -0.1324

----------------------------------------
Additional Information:
The number in degrees: 1432.39°
Cosine (in radians): cos(25.0) = 0.9912
Tangent (in radians): tan(25.0) = -0.1335

Requirements
- Python 3.x
- No external libraries required (math module is built-in)

Key Concepts Demonstrated:
- Function definition and calling
- Using built-in Python modules (math module)
- Parameter passing and return values
- Error handling with try-except blocks
- String formatting for clean output
